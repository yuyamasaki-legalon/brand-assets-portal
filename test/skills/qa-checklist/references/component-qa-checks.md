# コンポーネント別 QA チェック項目

Key Components に記載されたコンポーネントに対応する QA チェック項目集。
仕様のコンテキストに合わせてカスタマイズして使用する。

---

## データ表示

### Table / DataTable

- [ ] データが正しく表示される（列の内容・順序）
- [ ] ソート操作で昇順/降順が切り替わる
- [ ] ページネーションが正しく動作する（次/前/指定ページ）
- [ ] 行クリック/選択が正しく動作する
- [ ] 空データ時に EmptyState が表示される
- [ ] Loading 時に Skeleton.Table が表示される
- [ ] カラムヘッダーが仕様通りのラベル

### Card

- [ ] カード内のコンテンツが仕様通りに表示される
- [ ] クリック可能な場合、クリック動作が正しい
- [ ] ホバー時のスタイル変化（クリック可能な場合）

### List

- [ ] リスト項目が正しく表示される
- [ ] 空リスト時の表示が適切
- [ ] スクロール動作が正しい

---

## フォーム入力

### TextField

- [ ] 入力値が正しく反映される
- [ ] プレースホルダーが表示される
- [ ] 必須フィールドのラベル表示
- [ ] バリデーションエラー時に FormControl のエラー表示
- [ ] 最大文字数制限が動作する（設定時）
- [ ] disabled 状態で入力不可

### Select

- [ ] 選択肢が正しく表示される
- [ ] 選択変更が正しく反映される
- [ ] デフォルト値が正しく設定される
- [ ] プレースホルダーが表示される（未選択時）
- [ ] disabled 状態で操作不可

### Combobox

- [ ] 入力によるフィルタリングが動作する
- [ ] 選択肢が正しく表示される
- [ ] 該当なし時の Empty 表示
- [ ] Loading 時のインジケーター表示
- [ ] 選択後の表示が正しい

### DateField / RangeDateField

- [ ] 日付選択が正しく動作する
- [ ] 不正日付の入力がバリデーションされる
- [ ] 日付フォーマットが正しい
- [ ] RangeDateField の場合、開始日 > 終了日の制御

### Textarea

- [ ] 複数行入力が正しく動作する
- [ ] リサイズ動作が適切
- [ ] 最大文字数制限が動作する（設定時）

### NumberField

- [ ] 数値のみ入力可能
- [ ] 最小値/最大値の制御
- [ ] ステッパー操作が正しい（ある場合）

### TagPicker

- [ ] タグの追加・削除が動作する
- [ ] 検索によるフィルタリングが動作する
- [ ] 選択済みタグが正しく表示される

### Switch / Checkbox

- [ ] ON/OFF の切り替えが正しく動作する
- [ ] 状態変更が即座に反映される
- [ ] ラベルが仕様通り

### RadioGroup

- [ ] 各選択肢が正しく表示される
- [ ] 排他的選択が動作する
- [ ] デフォルト選択が正しい

---

## ナビゲーション・構造

### Tab.Group / Tab.List

- [ ] 各タブが正しいラベルで表示される
- [ ] タブ切り替えで対応するコンテンツが表示される
- [ ] デフォルト選択タブが正しい
- [ ] キーボード操作（矢印キー）でタブ切り替えが可能

### Sidebar / SidebarNavigation

- [ ] ナビゲーション項目が正しく表示される
- [ ] アクティブ項目のハイライトが正しい
- [ ] クリックで対応ページに遷移する

### ContentHeader

- [ ] タイトルが仕様通り
- [ ] アクションボタンが正しく表示される
- [ ] パンくずリスト（ある場合）が正しい

---

## オーバーレイ

### Dialog

- [ ] ダイアログが正しく開閉する
- [ ] ESC キーで閉じることができる
- [ ] 外側クリックでの閉じる動作（仕様に依存）
- [ ] フォーカストラップが動作する（Tab キーがダイアログ内に留まる）
- [ ] DialogHeader のタイトルが正しい
- [ ] アクションボタン（保存/キャンセル等）が正しく動作する

### Drawer

- [ ] Drawer が正しく開閉する
- [ ] ESC キーで閉じることができる
- [ ] Drawer 内のコンテンツが正しく表示される
- [ ] Drawer 内でのスクロール動作
- [ ] 閉じた後のフォーカス復帰

### Popover

- [ ] トリガー操作（クリック/ホバー）で表示される
- [ ] コンテンツが正しく表示される
- [ ] 外側クリックで閉じる

---

## フィードバック

### Button

- [ ] クリックで期待されるアクションが実行される
- [ ] loading 状態でスピナーが表示され、操作不可になる
- [ ] disabled 状態で操作不可
- [ ] ラベルが仕様通り

### IconButton

- [ ] クリックで期待されるアクションが実行される
- [ ] Tooltip でラベルが表示される（ホバー時）
- [ ] aria-label が設定されている

### Banner

- [ ] 適切な色（danger/warning/info）で表示される
- [ ] メッセージが仕様通り
- [ ] 閉じるボタン（ある場合）が動作する

### Snackbar

- [ ] 操作後に正しいメッセージが表示される
- [ ] 適切な色（success/danger）で表示される
- [ ] 自動で消える（タイムアウト）
- [ ] アクションボタン（ある場合）が動作する

---

## 表示

### StatusLabel

- [ ] ステータスに応じた色で表示される
- [ ] ラベルテキストが正しい

### Badge

- [ ] カウント/ドットが正しく表示される
- [ ] 0件時の非表示動作（仕様に依存）

### Avatar

- [ ] ユーザー画像/イニシャルが正しく表示される
- [ ] サイズが適切

### EmptyState

- [ ] 適切なサイズ（large/medium/small）で表示される
- [ ] イラスト/アイコンが適切
- [ ] メッセージが仕様通り
- [ ] アクションボタン（ある場合）が動作する

### Skeleton

- [ ] データ読み込み中に表示される
- [ ] データ取得後に消えて実コンテンツが表示される
- [ ] レイアウトが実コンテンツと一致

---

## レイアウト

### PageLayout

- [ ] レイアウトパターンが仕様通り
- [ ] 各エリア（Sidebar/Main/Pane/Footer）の配置が正しい
- [ ] レスポンシブ動作（必要な場合）

### Toolbar

- [ ] ツールバー内のアクションが正しく配置される
- [ ] 各ボタン/コントロールが動作する

### Search

- [ ] 検索入力が動作する
- [ ] 検索結果が正しくフィルタリングされる
- [ ] クリアボタンが動作する
