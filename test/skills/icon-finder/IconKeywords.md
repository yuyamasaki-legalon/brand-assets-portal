# アイコンキーワードデータベース

Aegis アイコンのセマンティック検索用キーワード一覧。
AI がアイコンを選択する際に、キーワードで Grep 検索して適切なアイコンを見つける。

## 使用方法

```bash
# 例: 「検索」に関連するアイコンを探す
rg "検索" skills/icon-finder/IconKeywords.md
```

`rg` が使えない場合は `grep` を使う:

```bash
grep "検索" skills/icon-finder/IconKeywords.md
```

## 検索のヒント

- **日本語と英語の両方を試す**: 例: 「検索」「search」「サーチ」
- **類義語を考慮する**: 例: 「削除」→「ゴミ箱」「trash」
- **動作と形状の両方**: 例: 「拡大」は虫眼鏡か最大化か

## 使用例

```tsx
import { LfMagnifyingGlass, LfTrash, LfPlusLarge } from "@legalforce/aegis-icons";
import { Icon } from "@legalforce/aegis-react";

// 検索アイコン
<Icon><LfMagnifyingGlass /></Icon>

// 削除アイコン
<Icon><LfTrash /></Icon>

// 追加アイコン
<Icon><LfPlusLarge /></Icon>
```
