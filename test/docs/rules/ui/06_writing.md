# 6. ライティング

この章では、UI上のテキスト（ライティング）に関するルールを定義します。

## ガイドライン

UI上のライティングに関する詳細なルールは、以下の公式ドキュメントで定義されています。UI上のテキストを作成・レビューする際は、必ず関連するドキュメントを参照してください。

> **基本的な考え方**
>
> *   プロダクトのトーン＆マナーや、英語翻訳時の注意点など、すべてのライティングの基礎となる原則を定義しています。
> *   [サービスUIテキストガイドライン](https://www.notion.so/1a0316695712807c908cf46a3752fcd0)

> **用字・用語**
>
> *   漢字とひらがなの使い分け、カタカナ語の表記、日時や単位の書き方など、具体的な単語の表記ルールを定義しています。
> *   [サービスUIテキストガイドラインのビュー](https://www.notion.so/1a03166957128008834dd52e48745201)
> *   [用語索引（ルールの逆引き用）](https://www.notion.so/1a0316695712805196a2e58c4d6015da)

> **UIテキスト**
>
> *   ボタンラベル、エラーメッセージ、プレースホルダーなど、各UIコンポーネントに表示するテキストの具体的なパターンと実例を定義しています。
> *   [サービスUIテキストガイドラインのビュー](https://www.notion.so/1a031669571280d48800dcb7998e263e)
