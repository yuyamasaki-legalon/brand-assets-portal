> ## 件数の表示
>
> ### 基本方針
>
> * **【指針】** ユーザーは表示中の数が、現在表示中のアイテム以外にもN個あることが表記により理解できる。
> * **【指針】** ユーザーは表示中の数が、現在表示中のアイテムを含めN個あることが表記により理解できる。
>
> ### 事例
>
> * **表示中の数を超える場合の表現:** `数字+` （+は数字の後）
>     * 例: バッジの表示上限が99件のフィルターに100件の設定がされている → `99+`
> * **他にデータがある場合の表現:** `+数字` （+は数字の前）
>     * 例: 担当者が4人いる → `田中太郎 +3`
> * **【ルール】** 件数を短縮表示する場合は超過の場合に+を使用する。
>     * OK: 検索結果が101件の時に`100+`件と表示する
>     * NG: 検索結果が100件の時に`100+`件と表示する

> ### 選択コントロールの使い分け
>
> #### チェックボックス
>
> * **【指針】** 複数の選択肢の中から、任意（0個以上）の項目を選択できる場合に使用します。
> * **【指針】** 項目が多数ある場合は「すべて選択／すべて解除」機能の設置を検討します。
> * **【指針】** 選択肢が親子関係を持つ場合は、親の選択・解除が子にどう影響するかの挙動を定義します。（例: 親をチェックしたら子はすべてチェックされる）
