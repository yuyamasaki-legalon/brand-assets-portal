# 使用時の注意点
Figmaにのみ存在するコンポーネントです。<br>実装上は単にActionListとして利用します。<br>
Menu内での使用については<mention-page url="https://www.notion.so/15831669571280f8a546da0fe93fda1f"/>を参照してください
PageLayoutに常駐するものとして使用する場合<mention-page url="https://www.notion.so/15831669571280f8a546da0fe93fda1f"/>の使用を検討してください。<br>リストの内容が固定ではなく、動的に変わる場合にのみ使用してください。
