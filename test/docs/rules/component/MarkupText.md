# MarkupText

💡 **MarkupTextは、RichTextEditorの中や、Slack連携時の本文表示などで使用するコンポーネントです。<br>テキストを視覚的に区別し、情報の伝達を補助します。**<br>
	figmaでは以下を同一のコンポーネントとして扱っています。
	- [**OrderedList**](https://www.chromatic.com/component?appId=634618c98b322242afd2ae10&csfId=components-orderedlist&buildNumber=3277&k=67580af56f0662716dbc89fd-1200px-interactive-true&h=1&b=-1)
	- [**UnorderedList**](https://www.chromatic.com/component?appId=634618c98b322242afd2ae10&csfId=components-unorderedlist&buildNumber=3277&k=67580af56f0662716dbc8ad0-1200px-interactive-true&h=1&b=-1)
	- [**CodeBlock**](https://www.chromatic.com/component?appId=634618c98b322242afd2ae10&csfId=components-codeblock&buildNumber=3277&k=67580af56f0662716dbc896f-1200px-interactive-true&h=1&b=-1)
	- [**Code**](https://www.chromatic.com/component?appId=634618c98b322242afd2ae10&csfId=components-code&buildNumber=3277&k=67580af56f0662716dbc896d-1200px-interactive-true&h=1&b=-1)
	- [**Blockquote**](https://www.chromatic.com/component?appId=634618c98b322242afd2ae10&csfId=components-blockquote&buildNumber=3277&k=67580af56f0662716dbc893c-1200px-interactive-true&h=1&b=-1)

# 使用時の注意点
<columns>
	<column>
		<span color="red">**⚠️RichiTextEditorの入力、または入力後の本文のみに使用してください。<br>文中の単なる装飾としての利用はできません。**</span>
		リストに中黒\[・\]や数字\[1,2,3\]が必要な場合はfigmaのテキスト編集で行ってください。<br>また、エンジニアには`ul > li` で組んでもらい、`list-style-type`で調整してもらうように伝えてください。
