# HorizonScrollButton

💡 **HorizonScrollButtonは、Tableなどで、画面内に収まらない際に自動的に表示されるコンポーネントです。**

# 使用時の注意点
Tableの内容が一画面に収まらない場合に自動的表示されるTable固有のコンポーネントです。<br>実装上はTableに含まれており、単独使用するすることはありません。
