# 使用時の注意点

### TableActionCell内のButtonについて
できる限りラベルがあるButtonを用いてください。<br>IconButtonを用いる場合は認知可能なものだけにしてください。<br><br>ラベル付きボタンでアイコンを使用する際は<br>追加=[+]<br>削除=[🗑️]<br>のような判断がつきやすいものを使用するようにしてください。<br>難解なアイコンであれば無理にアイコンをつける必要はありません。
