# 使用時の注意点
<columns>
	<column>
		<synced_block url="https://www.notion.so/15831669571280b5a52eea72fec21ecc#17c316695712807a9f5dd13f962b33e0">
			⚠️<span color="red">**長文になる可能性がある文章にのみ使用してください。**</span>
			２、３行に収まる文章量の場合、かつ、その文字数が増減しない場合はBodyコンポーネントを利用してください。
		</synced_block>

	</column>
	<column>
		
	</column>
</columns>
**Document Bodyを使用するべきUIは以下のようなものです。**
- メール文面の表示
- Slack文面の表示
- メモ（特記事項）の表示
- コメントの表示
- 契約書や、書類の本文表現
- チャット本文の表現
上記は画面上の表示に限定します。<br>入力の際のTextareaにおけるテキストのスタイルはコンポーネントの初期値から変更しないでください。
