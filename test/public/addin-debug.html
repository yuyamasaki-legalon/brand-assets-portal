Sample Document

This area simulates the Word document editing surface. The task pane on the right loads the aegis-lab app inside an iframe, replicating the Office Add-in environment.

The iframe triggers window.self !== window.top detection, so HashRouter is automatically selected — the same behavior as in a real Office Add-in.

Task pane is resizable: drag the left edge of the panel.
Use the preset buttons (300 / 320 / 480) in the task pane header bar.
