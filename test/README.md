# Aegis Lab

**プロトタイプ量産のための実験場**

AI との対話で「言葉」から高精度な画面を生成し、実際に動く UI を爆速で試行錯誤できます。

🌐 **公式サイト**: https://aegis-lab.ontechnologies.tech/ （社内ネットワーク必須）

📖 **[aegis-lab について詳しく](./docs/about.md)** - なぜ aegis-lab なのか、他ツールとの違い

## クイックスタート

初めての方は **[オンボーディングガイド](./docs/onboarding-guide.md)** から始めてください。

## よく使うコマンド

| コマンド | 説明 |
|---------|------|
| `pnpm dev` | 開発サーバー起動 |
| `pnpm build` | ビルド（PR前必須） |
| `pnpm format` | リント・フォーマット |

→ [全コマンド一覧](./docs/commands.md)
