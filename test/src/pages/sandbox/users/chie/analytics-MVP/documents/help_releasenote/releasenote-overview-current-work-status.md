# 本機能を提供することによって得られる効果

マターマネジメントで対応中・完了した案件データを活用したレポート機能をリリースします。この機能により、主に法務管理者の皆様に以下の効果を提供します。

- 未割り当て案件数と担当者別の負荷を一画面で可視化し、担当者アサインの判断を迅速化します。
- 納期・ステータス・案件タイプ別の内訳から、案件の滞留状況や法務担当者の業務の偏りを把握できます。
- 納期超過を抱える担当者を即座に検知し、早期のリソース調整とリスク対応を可能にします。

# 前提

本機能は2月末~3月末にかけて、当初予定していた機能を何段階かに分けてリリースを行います。
本リリースは、その中の第一弾の「現在の業務状況」タブのリリースになります。

## 概要

「未割り当て案件が何件あるか」「誰が今どれだけ案件を抱えているか（納期・ステータス・種類別）」が一画面で分かり、納期超過している人も一目で分かる。誰にどの案件を振るかの判断や、負荷の偏りの把握に使える機能です。

## 機能詳細

### 未割り当ての案件Card

1. **未割り当て案件数の確認**
    - 主担当が決まっていない案件が何件あるかを確認できます。
    - 集計対象の設定（主担当のみ／副担当のみ／主＋副）に応じて、副担当が決まっていない案件数も併せて確認できます。
2. **案件一覧への遷移（アサイン作業への誘導）**
    - 「案件を確認」ボタンをクリックすると、**案件一覧画面が別タブで開き**、未割り当ての案件が自動的に絞り込まれた状態で表示されます。
    - 集計対象の設定（主担当のみ／副担当のみ／主＋副）に応じて、主担当または副担当が決まっていない案件一覧に遷移も併せて確認できます。

### 担当者別Card

1. **メンバーごとの「いま抱えている案件」の全体像の把握**
    - **誰が何件の進行中案件を持っているか**が一覧で分かります。
    - **納期ごとの内訳**: 納期超過・今日まで・3日以内・1週間以内・1週間以降・未入力の件数内訳。
    - **ステータスごとの内訳**: 未着手・対応中・レビュー中などの件数内訳。
    - **案件タイプごとの内訳**: 契約書審査・契約書起案・法務相談などの件数内訳。
2. **リスクの可視化**
    - **納期超過案件を抱えているメンバー**の名前が赤字＋警告アイコンで表示されます。
3. **メンバーを選んだときの「中身」の確認**
    - 特定メンバーの名前をクリックすると、その人の**進行中案件の一覧**（案件名・納期・ステータスなど）が見えます。件数だけでなく、「具体的にどの案件を抱えているか」まで確認できます。
4. **表示の切り替えと絞り込み**
    - **表示軸の切替**: メンバー別の案件内訳を「納期別」「ステータス別」「案件タイプ別」に切り替えて表示できます。
    - **フィルター**: 納期・ステータス・案件タイプで表示する案件を絞り込めます（例：「今日までの案件」だけを表示）。
    - **表示形式の切替**: 視覚的な「グラフ」と、詳細な数値が見える「テーブル」を切り替えられます。
    - **ソート**: 案件数順（忙しい順）やメンバー名順で並び替えができます。

### その他の操作
- **メンバー選択**: 特定のメンバーやユーザーグループに絞って表示できます。
- **集計対象の切替**: 「主担当のみ」「副担当のみ」「主担当＋副担当」を切り替えて集計できます。
