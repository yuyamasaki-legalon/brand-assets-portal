# Pane - Overview Tab - Card 2: 案件の対応時間

## 機能仕様
### カード概要

- 目的: 個人の月次リードタイム・初回返信速度の推移を確認する
- カード名: 案件の対応時間
- 対象ユーザー: Pane で選択中の 1 ユーザー
- 情報アイコン: `common-functional-spec.md`「画面・UI共通仕様 / カード共通UI」参照

### 表示・操作

- 集計期間:
  - メインコンテンツから引き継いだ集計期間を初期値として使用する
  - 指標定義は `terms-and-definitions.md`「リードタイム・初回返信の定義」参照
- 集計期間表示: カードヘッダーに表示する（UIは `common-functional-spec.md`「画面・UI共通仕様 / カード共通UI」参照）
- フィルター: `案件タイプ / 集計期間`
- フィルターUI: `common-functional-spec.md`「画面・UI共通仕様 / カード共通UI」参照
- 表示形式: `グラフ / テーブル`
- 初期表示: `グラフ`

### グラフ表示

- グラフ種別: 複合線グラフ
- 軸: X軸は月、Y軸は日数
- 系列: `リードタイム（中央値） / 初回返信速度（中央値）`
- 補助表示: 選択期間全体の中央値カードをグラフ上部に表示する
- X軸月ラベル操作: `common-functional-spec.md`「グラフ表示の共通ルール」参照

### テーブル表示

- カラム構成: `月（固定） / リードタイム中央値 / 初回返信速度中央値`
- 行構成: `全期間行` + `月別行`
- 行順: `全期間行` を先頭、その後 `月別行` を時系列昇順で表示する
- カラム表示形式: `common-functional-spec.md`「テーブルビューの共通ルール」参照
- ソート: `common-functional-spec.md`「SORT-COMMON-001」参照（グラフ/テーブル共通）

### 集計・データルール

- リードタイム定義: `terms-and-definitions.md`「リードタイム・初回返信の定義」参照
- 初回返信速度定義: `terms-and-definitions.md`「リードタイム・初回返信の定義」参照
- 欠損/0件・月次表示順: `terms-and-definitions.md`「月次集計の定義」参照

### 検証観点

- `PANE-LEADTIME-CARD-VAL-001` 初期表示はグラフで、2系列（リードタイム中央値/初回返信速度中央値）を表示する
- `PANE-LEADTIME-CARD-VAL-002` 表示形式切替時、同一集計値をグラフ/テーブルで一致表示する
- `PANE-LEADTIME-CARD-VAL-003` テーブル表示時、`全期間行` と `月別行` を表示する
- `PANE-LEADTIME-CARD-VAL-004` 該当月に対象案件がない場合は欠損表示ルールに従う
- `PANE-LEADTIME-CARD-VAL-005` 集計期間変更時、対象月のみで再集計する
