# 現在の業務状況を確認する

このページでは、チームメンバーの現在の業務負荷や未割り当て案件の状況を確認できる「現在の業務状況」タブについて説明しています。

**1.** 現在の業務状況タブとは？ **2.** 未割り当て案件の確認 **3.** 担当者別案件状況の確認 **4.** フィルター・表示切替機能 **5.** 詳細の確認 **6.** 活用例

### 1. 現在の業務状況タブとは？

現在の業務状況タブは、チーム全体の進行中案件の状況を可視化する機能です。「未割り当て案件が何件あるか」「誰が今どれだけ案件を抱えているか」を一画面で把握でき、納期超過などのリスクも早期に発見できます。これにより、適切なアサイン判断や負荷分散をサポートします。

### 2. 未割り当て案件の確認

画面上部の「未割り当ての案件」カードでは、主担当が決まっていない進行中案件の件数を確認できます。

*   **件数の確認**: 主担当（および設定により副担当）が未設定の案件数が表示されます。
*   **案件一覧への遷移**: 「案件を確認」ボタンをクリックすると、案件一覧画面が別タブで開き、未割り当て案件のみが絞り込まれた状態で表示されます。これにより、即座にアサイン作業を行うことができます。

### 3. 担当者別案件状況の確認

「進行中の案件状況」カードでは、各メンバーが現在抱えている案件の全体像をグラフやテーブルで確認できます。

*   **案件数の内訳表示**:
    *   **納期別**: 納期超過・今日まで・3日以内・1週間以内・1週間以降・未入力の件数内訳。
    *   **ステータス別**: 未着手・対応中・レビュー中などのステータスごとの件数内訳。
    *   **案件タイプ別**: 契約書審査・契約書起案・法務相談などの案件タイプごとの件数内訳。
*   **リスクの可視化**: 納期超過案件を抱えているメンバーは、名前が赤字＋警告アイコンで表示されます。誰の業務が逼迫しているか（「火の車」状態か）を一目で把握できます。

### 4. フィルター・表示切替機能

画面上部の操作エリアやカード内の機能を使って、表示内容をカスタマイズできます。

*   **メンバー選択**: 特定のメンバーやユーザーグループに絞って表示できます。
*   **集計対象の切替**: 「主担当のみ」「副担当のみ」「主担当＋副担当」を切り替えて集計できます。
*   **表示軸の切替**: メンバー別の案件内訳を「納期別」「ステータス別」「案件タイプ別」に切り替えて表示できます。
*   **フィルター**: 納期・ステータス・案件タイプで表示する案件を絞り込めます（例：「今日までの案件」だけを表示するなど）。
*   **表示形式の切替**: 視覚的な「グラフ」と、詳細な数値が見える「テーブル」を切り替えられます。
*   **ソート**: 案件数順（忙しい順）やメンバー名順で並び替えができます。

### 5. 詳細の確認

グラフの棒やメンバー名をクリック、またはテーブルの「開く」ボタンをクリックすると、画面右側に**メンバー詳細Pane**が開きます。
ここでは、選択したメンバーが抱えている進行中案件の一覧（案件名・納期・ステータスなど）を確認でき、件数だけでなく「具体的にどの案件を抱えているか」まで把握できます。

### 6. 活用例

現在の業務状況タブは以下のようなシーンでご活用いただけます。

*   **未割り当て案件のアサイン**
    *   未割り当て案件数を確認し、「案件を確認」ボタンから一覧へ遷移して担当者を決定する。
*   **チームの負荷分散**
    *   メンバーごとの案件数や納期状況を比較し、特定のメンバーに負荷が偏っていないかを確認する。余裕があるメンバーを見つけて、案件の再配分を検討する。
*   **リスク管理**
    *   納期超過案件を抱えているメンバーを特定し、詳細Paneで具体的な案件を確認してフォローを行う。
