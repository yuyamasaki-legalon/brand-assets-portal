# Volume Tab Spec

## 機能仕様
### 案件数タブ

- 目的: チーム全体の業務負荷を、期間別・ユーザー別に把握する。
- 主要指標:
   - 月次の新規案件数
   - 月次の完了案件数
   - ユーザー別の完了案件数

### 上部入力（タブ共通）

- ユーザー・ユーザーグループ:
   - UI 仕様は `common-functional-spec.md`「画面・UI共通仕様 / カード共通UI」参照。
   - 反映範囲: タブ内の全カード。
- 集計期間:
   - UI 仕様は `common-functional-spec.md`「画面・UI共通仕様 / カード共通UI」参照。
   - 初期値: 直近 6 か月（今月を含む）。
   - 期間制約: 開始月から 12 か月以内。
   - 反映範囲: タブ内の全カード。
- リセットボタン:
   - 表示条件: ユーザー・ユーザーグループまたは集計期間が初期値と異なる場合に表示。
   - 動作: ユーザー・ユーザーグループを未選択、集計期間を直近 6 か月に戻す。

### タブ共通ルール

- 上部入力の変更は、カード 1〜3 の表示・集計に即時反映する。
- 上部入力のリセットは、カード 1〜3 の表示・集計に即時反映する。
- 集計定義・用語は `terms-and-definitions.md` を参照する。

### 検証観点

- `VOLUME-TAB-VAL-001` 初期表示時、ユーザー・ユーザーグループは未選択である。
- `VOLUME-TAB-VAL-002` 初期表示時、集計期間は直近 6 か月である。
- `VOLUME-TAB-VAL-003` 上部入力変更時、タブ内の全カード表示・集計が更新される。
- `VOLUME-TAB-VAL-004` リセット実行時、上部入力が初期値に戻る。
