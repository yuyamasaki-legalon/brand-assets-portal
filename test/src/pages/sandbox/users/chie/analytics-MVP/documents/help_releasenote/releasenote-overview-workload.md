# 本機能を提供することによって得られる効果

マターマネジメントで新規に流入した案件や完了した案件などの「案件数」に着目したレポート機能をリリースします。この機能により、主に法務管理者の皆様に以下の効果を提供します。

- 新規案件と完了案件の推移を可視化し、チーム全体の業務量トレンドや繁忙期・閑散期を把握できます。
- メンバーごとの完了案件数を比較し、パフォーマンスのばらつきや育成が必要なポイントを発見できます。
- 納期遵守率などの指標を通じて、法務サービスの品質（SLA）を定量的に評価できます。

# 前提

本機能は2月末~3月末にかけて、当初予定していた機能を何段階かに分けてリリースを行います。
本リリースは、その中の第二弾の「案件数」タブのリリースになります。

また、第一弾リリース（現在の業務状況タブ）で未実装だった、担当者別Cardの「依頼部署」軸での表示機能も本リリースに含まれます。

## 概要

レポート機能の案件数タブでは、「チーム全体の新規・完了案件の推移」と「メンバーごとの完了案件数」を一画面で可視化。誰がどれだけ案件を処理しているか、負荷が偏っていないか、処理速度に課題があるメンバーは誰かを把握し、業務改善や評価に活用できます。

## 機能詳細

### 新規案件Card

1. **新規案件の発生傾向の把握**
    - 月ごとの新規案件数の推移を確認し、繁忙期や閑散期を把握できます。
    - 案件タイプごとや依頼部署ごとの内訳を表示し、どの種類の案件が増えているか、どの部署からの依頼が増えているかを分析できます。
2. **昨年対比での分析**
    - 「昨年を表示」オプションを使用することで、前年同月と比較して案件が増加しているか減少しているかを確認できます。
3. **表示形式の切り替え**
    - **月次棒グラフ**: 月ごとの推移を確認するのに適しています。
    - **全期間円グラフ**: 指定期間全体での案件タイプの構成比を確認するのに適しています。
    - **テーブル**: 正確な数値を確認するのに適しています。

### 完了案件Card

1. **完了案件の質の確認**
    - 月ごとの完了案件数の推移を確認できます。
    - **案件タイプ別・依頼部署別**: どの種類の案件が多く処理されたか、どの部署からの依頼が多く処理されたかを確認できます。
    - **納期遵守状況**: 納期内に完了した案件、納期超過して完了した案件、納期未入力で完了した案件の内訳を確認し、チーム全体の納期遵守率を把握できます。

2. **昨年対比での分析**
    - 新規案件と同様に、前年同月との比較が可能です。
3. **表示形式の切り替え**
    - **月次棒グラフ**: 月ごとの推移を確認できます。
    - **全期間円グラフ**: 期間全体での納期遵守状況や案件タイプの構成比を確認できます。
    - **テーブル**: 詳細な数値を確認できます。

### 担当者別完了案件Card

1. **メンバーごとのパフォーマンス比較**
    - **完了案件数**: 誰が多くの案件を処理したかを比較できます。
2. **表示の切り替えと絞り込み**
    - **表示軸の切替**: 「案件タイプ別」「納期遵守状況別」「依頼部署別」に切り替えて内訳を確認できます。
    - **ソート**: 案件数順やメンバー名順で並び替えができます。
    - **フィルター**: 特定の案件タイプや納期状況で絞り込んで分析できます。
    - **表示形式の切替**: 視覚的な「グラフ」と、詳細な数値が見える「テーブル」を切り替えられます。
3. **主担当・副担当の確認**
    - 「主担当のみ」「副担当のみ」「主担当＋副担当」を切り替えて、それぞれの役割での貢献度を確認できます。

### その他の操作
- **メンバー選択**: 特定のメンバーやユーザーグループに絞って集計できます。
- **集計期間の設定**: 任意の期間（月単位）を指定してデータを集計できます。

### 現在の業務状況タブの担当者別Card（追加機能）

1. **依頼部署ごとの業務の偏りの把握**
    - 担当者別の進行中案件の内訳を「依頼部署別」に切り替えて表示できます。
    - 特定のメンバーが特定の部署からの依頼に偏っていないか、あるいは特定の部署からの依頼が誰に集中しているかを確認できます。
