# Pane - Ongoing Matters Tab Spec

## 機能仕様
### 進行中の案件タブ

- 対象案件: ステータスが完了以外すべて（未着手を含む）
- 一覧構成:
   - 主担当一覧 / 副担当一覧を分けて表示
   - 主担当一覧: 該当案件がある場合のみ表示（`0件` は非表示）
   - 副担当一覧: 該当案件がある場合のみ表示（`0件` は非表示）
   - 主担当・副担当とも `0件` の場合: EmptyState を表示
- 並び順: `納期未入力` を先頭、その後 `納期（日付）昇順`
- フィルター（`common-functional-spec.md`「画面・UI共通仕様 / カード共通UI」参照）:
   - 項目: `ステータス / 案件タイプ / 納期区分`
   - ステータス: テナント設定のステータスを全列挙（完了除く）
   - 納期区分: `すべて / 納期未入力 / 納期超過 / 今日まで / 今日含め3日以内 / 今日含め7日以内 / 1週間後〜`
   - 納期区分の定義: `terms-and-definitions.md`「納期区分（表示軸・フィルター）」参照
   - 初期値: すべて `すべて`
   - Pane起動時: メインコンテンツ（現在の業務状況）で設定済みの `案件タイプ / ステータス / 納期` を引き継ぐ
- 表示設定（カラム移動 / 表示非表示 / 幅調整 / 固定 / ソート）:
   - 保存・復元ルールは `common-functional-spec.md`「画面状態の保存ルール」参照
- 表示項目（最小）:
   - 共通: `案件名 / 納期 / ステータス（滞留日数付き）`
   - ステータス表示: ステータスラベル + 最後にステータスが変化してからの経過日数（滞留日数）
   - 滞留日数の定義: `terms-and-definitions.md`「滞留の定義」参照
   - 主担当一覧: 共通項目 + `副担当者名`
   - 副担当一覧: 共通項目 + `主担当者名`
   - 納期超過案件: 警告アイコン + `超過` ラベルで強調（定義: `terms-and-definitions.md`「納期超過の定義」参照）
- 行操作:
   - 行クリックで案件詳細ページへ遷移
   - 案件名列に「別タブで開く」アイコンボタンを表示
   - 遷移時・復帰時の状態保持は `common-functional-spec.md`「画面状態の保存ルール」参照
- 権限: `common-functional-spec.md`「権限・認可ルール」参照（閲覧可能案件のみ表示）
- 検証観点
   - `PANE-ONGOING-TAB-VAL-001` 主担当案件と副担当案件がそれぞれ対応する一覧に表示される。
   - `PANE-ONGOING-TAB-VAL-002` 主担当/副担当の表示切替は件数条件に従う（片側 `0件` は該当一覧のみ非表示、両側 `0件` は EmptyState）。
   - `PANE-ONGOING-TAB-VAL-003` 一覧は `納期未入力` 先頭 + `納期昇順` で表示される。
   - `PANE-ONGOING-TAB-VAL-004` フィルター適用時、選択条件に一致する案件のみ表示される。
   - `PANE-ONGOING-TAB-VAL-005` メインコンテンツから Pane を開いた場合、`案件タイプ / ステータス / 納期` 条件を引き継ぐ。
   - `PANE-ONGOING-TAB-VAL-006` 納期超過案件は視覚的に強調表示される。
   - `PANE-ONGOING-TAB-VAL-007` 行クリックと別タブ操作で案件詳細へ遷移できる。
   - `PANE-ONGOING-TAB-VAL-008` 遷移後に戻った際、フィルター・表示設定が復元される。
