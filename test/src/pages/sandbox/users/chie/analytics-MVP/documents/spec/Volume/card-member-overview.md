# Volume - Card 3: 完了案件（ユーザー別）

## 機能仕様
### カード概要

- 目的: ユーザー別の完了案件数の偏りを把握する
- カード名: 完了案件（ユーザー別）
- 情報アイコン: `common-functional-spec.md`「画面・UI共通仕様 / カード共通UI」参照
- 対象範囲: Volume タブの上部入力（ユーザー・ユーザーグループ、集計期間）の反映結果を対象とする

### 表示・操作

- 集計期間:
  - カード固有の集計期間が未設定の場合はタブ共通の集計期間を使用する
  - 集計期間フィルター初期値は未選択（タブ共通値を使用）とする
- 集計期間表示: カードタイトル下に表示する（UIは `common-functional-spec.md`「画面・UI共通仕様 / カード共通UI」参照）
- フィルター: `案件タイプ / 納期 / 集計期間`
- フィルターUI: `common-functional-spec.md`「画面・UI共通仕様 / カード共通UI」参照
- 集計対象: `主担当案件のみ / 副担当案件のみ / 主担当 + 副担当`
- 集計対象定義: `common-functional-spec.md`「集計対象の共通仕様」および `terms-and-definitions.md`「集計対象の設定」参照
- 表示切替: `案件タイプ別 / 納期別 / 依頼部署別`
- 表示切替初期値: `案件タイプ別`
- 表示形式: `グラフビュー / テーブルビュー`
- 初期表示: `グラフビュー`
- ソート項目: `案件数昇順 / 案件数降順 / ユーザー名昇順 / ユーザー名降順`
- 初期ソート: `案件数降順`
- ソート適用: グラフとテーブルの双方に同一の並び順を適用する
- リセット: カード内フィルター条件を初期化し、カード内集計期間はタブ共通の集計期間を再適用する

### グラフ表示

- グラフ種別: 積み上げ横棒グラフ
- 軸: X軸は案件数（件）、Y軸はユーザー
- 高さ: ユーザー数に応じて動的計算（最小240px、最大640px、超過時はスクロール）
- 凡例（納期別）: `納期内 / 納期超過 / 納期未入力`
- 凡例（案件タイプ別）: `契約書審査 / 契約書起案 / 法務相談 / その他`
- 集計対象が `主担当 + 副担当` または `副担当案件のみ` の場合、バーに `主 / 副` ラベルを表示する
- ユーザー名クリック: 対象ユーザーのPaneを開き、パフォーマンスタブに遷移する

### テーブル表示

- カラム構成: `ユーザー名（固定） / 合計 / 各内訳`
- 集計対象別ラベル: 内訳ラベルに `（主）/（副）` を付与する
- `開く` 操作: ユーザー名列の右端に表示し、対象ユーザーのPane（パフォーマンスタブ）を開く
- カラム表示形式: `common-functional-spec.md`「テーブルビューの共通ルール」参照
- ソート: `common-functional-spec.md`「SORT-COMMON-001」参照（グラフ/テーブル共通）

### 集計・データルール

- 完了案件数定義: `terms-and-definitions.md`「月次集計の定義」参照
- 納期判定: `terms-and-definitions.md`「納期超過の定義」参照
- 案件タイプフィルター適用時: 選択案件タイプのみ集計対象とする
- 納期フィルター適用時: 一致する案件のみ集計対象とする
- 表示切替`案件タイプ別`の区分定義: `terms-and-definitions.md`「用語・定義」参照
- 表示切替`納期別`の区分定義: `terms-and-definitions.md`「用語・定義」「納期超過の定義」参照
- 表示切替`依頼部署別`の区分定義: `terms-and-definitions.md`「用語・定義」参照
- 表示切替`依頼部署別`: 未対応（選択時はグラフ/テーブルを表示しない）

### 画面状態の保存

- 保存方針: `common-functional-spec.md`「画面状態の保存ルール」参照
- 保存対象: `案件タイプ / 納期 / 集計期間 / 集計対象 / 表示切替 / 表示形式 / ソート`

### 検証観点

- `VOLUME-MEMBER-CARD-VAL-001` 初期表示で `案件タイプ別`・`グラフビュー`・`案件数降順` を表示する
- `VOLUME-MEMBER-CARD-VAL-002` 表示切替変更時、選択軸（案件タイプ別/納期別）で内訳を更新する
- `VOLUME-MEMBER-CARD-VAL-003` `依頼部署別` 選択時は未対応表示（グラフ/テーブル非表示）となる
- `VOLUME-MEMBER-CARD-VAL-004` 集計対象が `主担当 + 副担当` の場合、主/副を区別して表示する
- `VOLUME-MEMBER-CARD-VAL-005` 表示形式切替後も同一集計値を維持する
- `VOLUME-MEMBER-CARD-VAL-006` ソート変更時、グラフ/テーブル双方で同一の並び順に更新される
- `VOLUME-MEMBER-CARD-VAL-007` テーブルの `開く` 操作で、対象ユーザーのPane（パフォーマンスタブ）を開く
- `VOLUME-MEMBER-CARD-VAL-008` 案件タイプ・納期・集計期間フィルター適用時、選択条件に一致する案件のみ集計する
- `VOLUME-MEMBER-CARD-VAL-009` リセット実行時、カード内フィルター条件を初期値に戻し、集計期間はタブ共通値を再適用する
- `VOLUME-MEMBER-CARD-VAL-010` グラフのY軸ユーザー名クリックで、対象ユーザーのPane（パフォーマンスタブ）を開く
