# レポート機能（現在の業務状況・案件数）を確認する

「レポート機能」の概要や基本的な使い方について記載します。

 

# レポート機能とは

マターマネジメントに蓄積された案件データを分析し、チームの状況を可視化する機能です。
レポート機能では、次のような業務を支援します。

- チーム全体の進行中案件の状況を可視化し、アサイン判断やリスク検知を行う
- 新規案件と完了案件の推移、メンバーごとの実績を可視化し、業務量のトレンド把握や評価を行う

※レポート機能で集計される案件は、閲覧しているユーザーが「閲覧権限以上」を持つ案件のみです。そのため、閲覧するユーザーによって表示される数値が異なる場合があります。

レポート機能は、以下のタブで構成されており、目的に応じて使い分けることで、適切なアサインや業務改善に役立てることができます。

- **現在の業務状況タブ**
  「今、誰が何を持っているか」「未割り当て案件はどれくらいあるか」といった**フロー（進行中）**の情報を確認します。
- **案件数タブ**
  「過去にどれくらい案件が来たか」「誰がどれくらい処理したか」といった**ストック（実績）**の情報を確認します。
 

# 現在の業務状況タブでできること

## フィルター・表示条件の切り替え（タブ全体）
- **メンバー選択**
  特定のメンバーやユーザーグループに絞ってデータを表示できます。
- **集計対象の切替**
  「主担当（案件担当者としてアサインされている案件）のみ」「副担当（副担当者としてアサインされている案件）のみ」「主担当＋副担当」を切り替えて集計できます。
  ※「主担当＋副担当」を選択した場合、1つの案件が主担当と副担当のそれぞれでカウントされます。

## 未割り当て案件の確認
- 主担当（および設定により副担当）が決まっていない進行中案件の件数を確認できます。
- 「案件を確認」ボタンから、未割り当て案件のみを絞り込んだ一覧画面へ直接遷移できます。
  ※「集計対象」の設定により、表示される件数や、遷移先の案件一覧で絞り込まれる条件（主担当未設定／副担当未設定）が変化します。

 

## 担当者別案件状況の確認
各メンバーが現在抱えている案件の全体像をグラフやテーブルで確認できます。

- **集計対象**
  各ユーザーが現在担当している、ステータスが完了状態以外の案件数を表示しています。

- **内訳の可視化**
  メンバーごとの案件数を「納期別」「案件ステータス別」「案件タイプ別」「依頼部署別」に切り替えて表示できます。

- **リスクの可視化**
  納期超過案件を抱えているメンバーは、名前が赤字＋警告アイコンで表示されます。誰の業務が逼迫しているかを確認できます。

- **詳細の確認・遷移**
  メンバー名をクリックすると、そのメンバーが抱えている進行中案件の一覧（案件名・納期・案件ステータスなど）を確認できます。
  一覧の案件名をクリックすると、該当案件の詳細画面へ遷移します。

# 案件数タブでできること

## フィルター・表示条件の切り替え（タブ全体）
- **メンバー選択**
  特定のメンバーに絞ってデータを表示できます。
- **集計期間の設定**
  任意の期間（月単位）を指定してデータを集計できます。
  ※デフォルトは、直近6ヶ月です。
  ※一度に選択できる期間は最大1年間です。

## 新規案件の分析

- **集計対象**
  指定期間内に作成された案件数を表示しています。

- **発生傾向の把握**
  月ごとの新規案件数の推移や、前年同月比を確認できます。

- **内訳の分析**
  表示軸を切り替えることで、どのような案件（案件タイプ別）や、どこからの依頼（依頼部署別）が増えているかを分析できます。
  ※指定期間全体の構成比を示す「円グラフ」も表示可能です。

- **昨年対比**
  「昨年を表示」オプションを使用することで、前年同月と比較して案件が増加しているか減少しているかを確認できます。

- **詳細の確認（ドリルダウン）**
  グラフの月をクリックすると、その月の新規案件一覧が別タブで開き、詳細を確認できます。

 

## 完了案件の分析

- **集計対象**
  指定期間内に案件ステータスが「完了」となった案件数を表示しています。
  ・複数回「完了」になった場合は、最後に変更された日時を基準にカウントします。
  ・現在の案件ステータスが「完了」の案件のみを集計します。

- **処理実績の確認**
  月ごとの完了案件数の推移を確認できます。

- **内訳の分析**
  表示軸を切り替えることで、以下の観点から内訳を確認できます。
  ※指定期間全体の構成比を示す「円グラフ」も表示可能です。
  - **納期遵守状況別**: 「納期内」「納期超過」「納期未入力」の割合から、チーム全体の納期遵守状況を把握できます。
  - **案件タイプ別・依頼部署別**: どの種類の案件やどの部署からの依頼が多く処理されたかを確認できます。

- **昨年対比**
  「昨年を表示」オプションを使用することで、前年同月と比較して完了案件が増加しているか減少しているかを確認できます。

- **詳細の確認（ドリルダウン）**
  グラフの月をクリックすると、その月の完了案件一覧が別タブで開き、詳細を確認できます。

 

## 担当者別完了案件の分析

- **集計対象**
  指定期間内に案件ステータスが「完了」となった案件のうち、各ユーザーが担当していた案件数を表示しています。
  ・複数回「完了」になった場合は、最後に変更された日時を基準にカウントします。
  ・現在の案件ステータスが「完了」の案件のみを集計します。

- **パフォーマンス比較**
  メンバーごとの完了案件数を比較し、誰が多くの案件を処理しているかを確認できます。

- **多角的な評価**
  単なる件数だけでなく、納期超過の有無や、主担当・副担当それぞれの実績を切り替えて確認することで、多角的な評価が可能です。

- **詳細の確認（ドリルダウン）**
  グラフやメンバー名をクリックすると、そのメンバーの完了案件一覧が右側に表示されます。一覧から各案件の詳細画面へ遷移することも可能です。

 

# 共通機能（フィルター・表示切替）
両方のタブで共通して利用できる操作です。

- **ソート（並び替え）**
  案件数順（昇順・降順）やメンバー名順（昇順・降順）で表示を並び替えることができます。

- **表示形式の切替**
  視覚的な「グラフ」と、詳細な数値が見える「テーブル」を切り替えられます。

- **各種フィルター**
  以下の項目で表示データを絞り込めます。
  ※タブやカードによって利用できるフィルター項目は異なります。
  - **案件タイプ**: 契約書審査、法律相談など
  - **案件ステータス**: 未着手、対応中、レビュー中など
  - **納期**: 今日まで、今週中、納期超過など
 

# 活用シーン
- **毎朝の状況確認（現在の業務状況タブ）**
  未割り当て案件がないか、納期超過しそうなメンバーがいないかを確認し、その日のアサインやフォローを行う。

- **月次の振り返り（案件数タブ）**
  先月の新規案件数や完了案件数を前年と比較し、業務量の増減傾向を把握する。
  特定の部署からの依頼が急増していないかを確認し、事業部への連携や体制強化を検討する。

- **メンバーの評価・1on1（両タブ）**
  完了案件数（量）と納期遵守状況（質）を定量的に確認し、フィードバックの根拠とする。
  特定のメンバーに負荷が偏っていないかを確認し、業務分担の見直しを行う。
