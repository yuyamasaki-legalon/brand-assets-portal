# Pane - Overview Tab - Card 1: 完了案件

## 機能仕様
### カード概要

- 目的: 個人の月次完了案件数の推移を確認する
- カード名: 完了案件
- 対象ユーザー: Pane で選択中の 1 ユーザー
- 情報アイコン: `common-functional-spec.md`「画面・UI共通仕様 / カード共通UI」参照

### 表示・操作

- 集計期間:
  - メインコンテンツから引き継いだ集計期間を初期値として使用する
  - 集計基準は `terms-and-definitions.md`「月次集計の定義」参照
- 集計期間表示: カードヘッダーに表示する（UIは `common-functional-spec.md`「画面・UI共通仕様 / カード共通UI」参照）
- フィルター: `案件タイプ / 集計期間`
- フィルターUI: `common-functional-spec.md`「画面・UI共通仕様 / カード共通UI」参照
- 表示形式: `グラフ（月次棒グラフ） / グラフ（全期間円グラフ） / テーブル`
- 初期表示: `グラフ（月次棒グラフ）`
- 表示軸: `納期別 / 案件タイプ別 / 依頼部署別`
- 表示軸初期値: `案件タイプ別`

### グラフ表示

- 月次表示: 積み上げ棒グラフ（X軸: 月、Y軸: 件数）
- 全期間表示: 円グラフ
- 凡例（納期別）: `納期内 / 納期超過 / 納期未入力`
- 凡例（案件タイプ別）: `契約書審査 / 契約書起案 / 法務相談 / その他`

### テーブル表示

- カラム構成: `月（固定） / 合計 / 各内訳`
- カラム表示形式: `common-functional-spec.md`「テーブルビューの共通ルール」参照
- ソート: `common-functional-spec.md`「SORT-COMMON-001」参照（グラフ/テーブル共通）

### 集計・データルール

- 完了案件数・欠損/0件・表示順: `terms-and-definitions.md`「月次集計の定義」参照
- 表示軸`納期別`の区分定義: `terms-and-definitions.md`「用語・定義」「月次集計の定義」参照
- 表示軸`案件タイプ別`の区分定義: `terms-and-definitions.md`「用語・定義」参照
- 表示軸`依頼部署別`の区分定義: `terms-and-definitions.md`「用語・定義」参照
- 表示軸`依頼部署別`: 未対応（選択時はグラフ/テーブルを表示しない）

### 検証観点

- `PANE-COMPLETED-CARD-VAL-001` 初期表示は `案件タイプ別` の月次棒グラフで表示する
- `PANE-COMPLETED-CARD-VAL-002` 表示軸切替時、選択した軸でグラフと内訳を更新する
- `PANE-COMPLETED-CARD-VAL-003` 表示形式切替時、同一集計値をグラフ/テーブルで一致表示する
- `PANE-COMPLETED-CARD-VAL-004` `依頼部署別` 選択時は未対応表示（グラフ/テーブル非表示）となる
- `PANE-COMPLETED-CARD-VAL-005` 集計期間変更時、対象月のみで再集計する
