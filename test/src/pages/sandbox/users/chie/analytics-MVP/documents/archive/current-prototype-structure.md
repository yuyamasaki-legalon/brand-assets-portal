# 現在のプロトタイプ構成

本ドキュメントは、`src/pages/sandbox/users/chie/analytics-MVP/` 配下で実装されているプロトタイプの現在の構成を記録したものです。

## タブ構成

### メインタブ（ページ全体）

ページ全体には2つのメインタブがあります。

1. **「進行中の案件」タブ** (`selectedTabIndex === 0`)
   - 現在進行中の案件の状況を把握するためのタブ
   - リアルタイムの業務状況を確認

2. **「パフォーマンス」タブ** (`selectedTabIndex === 1`)
   - 過去のデータを分析し、チームの生産性やメンバーの実績を確認するためのタブ
   - 共通フィルター（案件タイプ、メンバー、期間）が適用される

### 個人詳細パネル内のタブ

メンバー名をクリックして開く個人詳細パネル（`MemberDetailPane`）内にも2つのタブがあります。

1. **「進行中の案件」タブ** (`paneTabIndex === 0`)
   - 選択したメンバーの現在進行中の案件を表示

2. **「パフォーマンス」タブ** (`paneTabIndex === 1`)
   - 選択したメンバーの過去の実績を表示

## カード構成

### 「進行中の案件」タブ（メインタブ）

#### 1. 未割り当て案件カード（条件付き表示）

- **表示条件**: 主担当が未設定の案件がある場合、または副担当者が未設定の案件がある場合に表示
- **内容**:
  - 主担当が未設定の案件数
  - 副担当者が未設定の案件数（担当者モードが「副担当のみ」または「主担当 + 副担当」の場合）
- **機能**: 「案件一覧で確認」ボタンで未割り当て案件一覧を別タブで開く

#### 2. 「進行中の案件状況」カード

- **表示形式切り替え**: 
  - 納期別
  - ステータス別
  - 案件タイプ別
- **表示モード切り替え**: グラフ / テーブル
- **フィルター機能**:
  - 表示項目（主担当案件のみ / 副担当案件のみ / 主担当 + 副担当）
  - ステータス
  - 案件タイプ
  - 納期
- **機能**: 
  - メンバー名やグラフの各エリアをクリックすると個人詳細パネルが開く
  - ソート機能（案件数順 / ユーザー名順、昇順 / 降順）

### 「パフォーマンス」タブ（メインタブ）

#### 1. 「新規案件（案件タイプ別）」カード

- **表示内容**: 新規案件数のみ
- **期間切り替え**: 月別 / 全期間
- **表示形式切り替え**: 案件タイプ別 / 納期別
- **機能**:
  - 月別ビューでは棒グラフで表示
  - 全期間ビューでは円グラフで表示
  - 昨年対比表示の切り替え
  - 案件タイプフィルター

#### 2. 「完了案件」カード

- **表示内容**: 完了案件数のみ
- **期間切り替え**: 月別 / 全期間
- **表示形式切り替え**: 案件タイプ別 / 納期別
- **機能**:
  - 月別ビューでは棒グラフで表示
  - 全期間ビューでは円グラフで表示
  - 昨年対比表示の切り替え
  - 案件タイプフィルター

#### 3. 「案件の対応時間」カード（リードタイムグラフ）

- **表示内容**: 
  - リードタイム中央値
  - 初回返信速度中央値
- **期間切り替え**: 月別 / 全期間
- **機能**:
  - 月別ビューでは折れ線グラフで表示
  - 全期間ビューでは数値カードで表示
  - 昨年対比表示の切り替え
  - 案件タイプフィルター
  - 表示指標の選択（リードタイム中央値 / 初回返信速度中央値）

#### 4. 「完了案件（メンバー別）」カード（パフォーマンス概要）

- **表示内容**: メンバー別の完了案件数
- **表示形式切り替え**: 納期別 / 案件タイプ別
- **表示モード切り替え**: グラフ / テーブル
- **機能**:
  - グラフビューでは横棒グラフで表示
  - テーブルビューではメンバー別の詳細データを表示
  - 集計対象の選択（主担当のみ / 副担当のみ / 主担当 + 副担当）
  - 案件タイプフィルター
  - ソート機能（案件数順 / 名前順、昇順 / 降順）
  - メンバー名をクリックすると個人詳細パネルが開く

#### 5. 「案件の完了までの時間の内訳（メンバー別）」カード（リードタイム内訳）

- **表示内容**: 
  - リードタイムの内訳（待機時間、作業時間など）
  - 各メンバーの初回返信日数（`medianFirstReplyTime`）
- **表示モード切り替え**: グラフ / テーブル
- **機能**:
  - グラフビューでは横棒グラフ（積み上げ）で表示
  - テーブルビューではメンバー別の詳細データを表示
  - 集計対象の選択（主担当のみ / 副担当のみ / 主担当 + 副担当）
  - 案件タイプフィルター
  - ステータス分類設定（待機時間、作業時間などの分類をカスタマイズ可能）
  - ソート機能（案件数順 / 名前順、昇順 / 降順）
  - メンバー名をクリックすると個人詳細パネルが開く

### 個人詳細パネル内のカード構成

#### 「進行中の案件」タブ内

- **主担当案件一覧**: 選択したメンバーが主担当として担当している進行中の案件の一覧
- **副担当案件一覧**: 選択したメンバーが副担当として担当している進行中の案件の一覧
- **フィルター機能**: ステータス、案件タイプ、納期でフィルタリング可能

#### 「パフォーマンス」タブ内

- **「完了案件」カード**: 個人の完了案件数を表示
  - 期間切り替え: 月別 / 全期間
  - 表示形式切り替え: 納期別 / 案件タイプ別
  - 表示モード切り替え: グラフ / テーブル
  - 案件タイプフィルター
  - 期間フィルター

- **「案件の対応時間」カード**: 個人のリードタイムを表示
  - 期間切り替え: 月別 / 全期間
  - 表示モード切り替え: グラフ / テーブル
  - 案件タイプフィルター
  - 期間フィルター
  - 表示指標の選択（リードタイム中央値 / 初回返信速度中央値）
