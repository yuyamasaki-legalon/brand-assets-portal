# Current Workload Tab Spec

## 機能仕様
### 現在の業務状況タブ

### 上部入力（タブ共通）

- ユーザー・ユーザーグループ（`common-functional-spec.md` の `UI-COMMON-010, UI-COMMON-011` を参照）
- 集計対象（`common-functional-spec.md` の `ASSIGNEE-COMMON-001, ASSIGNEE-COMMON-002, ASSIGNEE-COMMON-003` を参照）
   - 表示: Selectコンポーネント
   - 配置: ユーザー・ユーザーグループフィルターの横
   - オプション: `主担当案件のみ / 副担当案件のみ / 主担当 + 副担当`
   - 初期値: 主担当案件のみ
   - 反映範囲: タブ内全体（未割り当ての案件カード、進行中の案件状況カード）の表示・集計対象に反映
     - ユーザーフィルターが適用されている場合、進行中の案件状況カードのy軸（ユーザー名）からも該当ユーザーが除外される
- リセットボタン
   - 表示: ユーザー・ユーザーグループフィルターまたは集計対象が初期値と異なる場合に表示
   - スタイル: `Button variant="subtle"`
   - 動作: クリックでユーザー・ユーザーグループフィルターを空に、集計対象を「主担当案件のみ」にリセット
- 検証観点
   - `CW-TAB-VAL-001` 集計対象の初期値は `主担当案件のみ` で表示する。
   - `CW-TAB-VAL-002` 集計対象変更時、タブ内の各カード表示・集計結果に反映する。
   - `CW-TAB-VAL-003` リセットボタンは、いずれかの条件が初期値と異なる場合のみ表示する。
   - `CW-TAB-VAL-004` リセット実行時、ユーザー・ユーザーグループフィルターと集計対象を初期値に戻す。
