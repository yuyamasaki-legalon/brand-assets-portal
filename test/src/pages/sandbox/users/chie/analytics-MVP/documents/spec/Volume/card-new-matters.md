# Volume - Card 1: 新規案件

## 機能仕様
### カード概要

- 目的: チーム全体の月次新規案件数の推移を俯瞰する
- カード名: 新規案件
- 情報アイコン: `common-functional-spec.md`「画面・UI共通仕様 / カード共通UI」参照

### 表示・操作

- 集計期間: タブ共通の集計期間設定を使用する
- 集計期間表示: カードヘッダーに表示する（UIは `common-functional-spec.md`「画面・UI共通仕様 / カード共通UI」参照）
- フィルター: `案件タイプ`
- フィルターUI: `common-functional-spec.md`「画面・UI共通仕様 / カード共通UI」参照
- 表示形式: `グラフ（月次棒グラフ） / グラフ（全期間円グラフ） / テーブル`
- 初期表示: `グラフ（月次棒グラフ）`
- 表示軸: `案件タイプ別 / 依頼部署別`
- 表示軸初期値: `案件タイプ別`
- 昨年表示: `昨年を表示` の ON/OFF で今年データ + 昨年同期データの同時表示を切り替える
- 昨年表示共通ルール: `common-functional-spec.md`「UI-COMMON-017」参照

### グラフ表示

- 月次表示: 積み上げ棒グラフ（X軸: 月、Y軸: 件数）
- 全期間表示: 円グラフ
- 凡例: `契約書審査 / 契約書起案 / 法務相談 / その他`
- X軸月ラベル操作: `common-functional-spec.md`「グラフ表示の共通ルール」参照

### テーブル表示

- カラム構成: `月（固定） / 合計 / 各案件タイプの新規案件数`
- 昨年表示ON時: `今年 / 昨年` を区別して表示する
- カラム表示形式: `common-functional-spec.md`「テーブルビューの共通ルール」参照

### 集計・データルール

- 新規案件数・欠損/0件・月次表示順: `terms-and-definitions.md`「月次集計の定義」参照
- 表示軸`案件タイプ別`の区分定義: `terms-and-definitions.md`「用語・定義」参照
- 表示軸`依頼部署別`の区分定義: `terms-and-definitions.md`「用語・定義」参照
- 表示軸`依頼部署別`: 未対応（選択時はグラフ/テーブルを表示しない）

### 検証観点

- `VOLUME-NEW-CARD-VAL-001` 初期表示は `案件タイプ別` の月次棒グラフで表示する
- `VOLUME-NEW-CARD-VAL-002` 表示軸切替時、選択した軸で表示を更新する
- `VOLUME-NEW-CARD-VAL-003` `昨年を表示` ON/OFF で今年/昨年の同時表示を切り替える
- `VOLUME-NEW-CARD-VAL-004` 表示形式切替時、同一集計値をグラフ/テーブルで一致表示する
- `VOLUME-NEW-CARD-VAL-005` 案件タイプフィルター適用時、選択条件に一致する案件のみ表示する
- `VOLUME-NEW-CARD-VAL-006` 月ラベル選択時、該当月の新規案件一覧を別タブで開く
- `VOLUME-NEW-CARD-VAL-007` `依頼部署別` 選択時は未対応表示（グラフ/テーブル非表示）となる
