# 案件ボールステータス表示機能

## サマリー

案件を事業部ボールか法務部ボールかわかるようにする機能。法務部が最終返信を実施した案件は自動的に事業部タスクに振り分けられる。

## 背景・課題

- **サービス**: LegalOn
- **モジュール**: マターマネジメント
- **要望種類**: 現在の機能の使い勝手を上げて欲しい

### 顧客の課題

- 進行中案件における責任の所在を明確にしたい
- 事業部ボールの案件も自身のタスクとして認識されてしまうためノイズになる

## 要件

### Confirmed（ユーザーから明示）

- [ ] 案件が事業部ボールか法務部ボールかを表示する
- [ ] 法務部が最終返信を実施した案件は、自動的に事業部タスクに振り分けられる

### Inferred（推定）

- [ ] 案件一覧にボールステータス列を追加
- [ ] タブで法務部ボール / 事業部ボール / すべてをフィルタリング
- [ ] セグメントカウンターでボールステータス別の件数を表示
- [ ] フィルター Drawer にボールステータスフィルターを追加
- [ ] 案件詳細 Drawer でボールステータスと最終返信者を表示
- [ ] 最終返信者が法務部の場合 → 事業部ボール、事業部の場合 → 法務部ボール

## 判定ロジック

| 最終返信者 | ボールステータス |
|-----------|----------------|
| 法務部（担当者） | 事業部ボール |
| 事業部（依頼者） | 法務部ボール |

## UI デザイン

- **法務部ボール**: `Tag variant="fill" color="blue"` で青色表示
- **事業部ボール**: `Tag variant="fill" color="neutral"` でニュートラル表示
- セグメントカウンターでボール別件数を視覚化
- タブで素早くフィルタリング可能
