# Accessibility (a11y) Requirements

## Mandatory
- `IconButton` → `aria-label` 必須 + `Tooltip` ラップ推奨
- `Dialog` → `DialogHeader` + `ContentHeader.Title` 必須
- `FormControl` → `FormControl.Label` 必須
- `Table` → `Table.Head` 必須
- `<div onClick>` → `role="button"` + `tabIndex={0}` + `onKeyDown` 必須
- `<img>` → `alt` 属性必須（装飾画像は `alt=""`）

## Dynamic Content
- ローディング Skeleton → `role="alert" aria-busy="true" aria-live="polite"` 必須

## スキル参照
アクセシビリティの包括的レビューが必要な場合 → `/a11y-audit` スキルを使用。
